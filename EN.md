# Poker Game - Hearts

[中文][tw] | [English][en]

* Four men poker game : Hearts
* Use Express + Socket io + Vue
* Run on [Heroku][Heroku]

## start the game

* Login (type your name)
* Create/join a room
* Game will start automatically after four members at room are all ready, have fun!
