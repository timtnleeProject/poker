# 西洋拱豬

[中文][tw] | [English][en]

* 四人連線撲克牌遊戲 : 西洋拱豬
* 使用 Express + Socket io + Vue
* 運行在 [Heroku][Heroku]

## start the game

* 登入 (輸入名字)
* 創建/加入房間
* 房間裡四名成員準備完成遊戲自動開始，玩得開心！
